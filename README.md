---

## The five Trust Service Criteria

| TSC | Code | What it covers |
|-----|------|----------------|
| Security | CC | Logical access, change management, risk, monitoring, vendor risk |
| Availability | A | Uptime commitments, monitoring, backup and recovery |
| Confidentiality | C | Data classification, NDAs, secure disposal |
| Processing Integrity | PI | Complete and accurate processing, error handling |
| Privacy | P | Privacy notice, consent, DSAR, retention, disclosure |

## How to use this

**90 days before fieldwork**
Run the Gap Assessment tab. Fix every High risk gap before your auditor arrives.

**30 days before fieldwork**
Work through the Evidence Tracker. Every item should be Complete before 
fieldwork starts. Run the Control Testing Log internally.

**During fieldwork**
Use the PBC Request Tracker to manage every auditor request.
Respond within 48 hours. Never miss a deadline.

**After fieldwork**
Update the Gap Assessment with any new findings.
Track all remediation actions with owners and due dates.
